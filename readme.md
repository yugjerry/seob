# Statistical Equilibrium of Optimistic Beliefs (SE-OB)

Code and experiments accompanying **Statistical Equilibrium of Optimistic Beliefs (SE-OB)** (Yu Gui and Bahar Ta≈ükesen, 2025).

- Paper (arXiv): https://arxiv.org/abs/2502.09569
- Repository contents:
  - `simu.py`: core implementations of SE-OB and logit QRE (fixed-point learning)

  - `convergence.ipynb`: convergence illustrations
